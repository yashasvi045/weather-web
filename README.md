
# Weather Web

A weather application web project built with modern web technologies.

## Overview

Weather Web is a responsive web application that provides weather information and forecasting capabilities. The project demonstrates best practices in web development with clean code structure and user-friendly interface.

## Features

- Real-time weather data
- Location-based forecasts
- Responsive design
- Clean and intuitive UI

## Technologies

- HTML5
- CSS3
- JavaScript

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Run `npm start`
4. Search city name and get the weather information

## Usage

Enter your location to view weather information.
